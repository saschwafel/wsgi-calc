Welcome to Schuyler's Calculator Program!

This program is designed to help you calculate simple values via the URL string.

Use the URL parameters to calculate numbers. localhost:8000/add/1/2 will add 1 and 2

Currently, the Calculator.py program supports

    Addition
    Subtraction
    Multiplication
    Division

For multiplication and division, anything past the first two operands (/1/3, for example) will be disregarded.

For addition and subtraction, all operands should be included. For subtraction, the 2nd, to the nth operands will be added, then subtracted from the first operand
